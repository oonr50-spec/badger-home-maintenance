BADGER HOME MAINTENANCE WEBSITE V2
==================================

This package is ready for GitHub Pages.

UPLOAD
1. Back up the current repository.
2. Keep your existing CNAME file if your custom domain already works.
3. Delete the old website files, but do not delete the repository itself.
4. Upload index.html, styles.css, script.js and the complete assets folder.
5. Restore the CNAME file at the top level.
6. Commit the changes and wait 2–5 minutes.
7. Open the website and press Ctrl + F5.
